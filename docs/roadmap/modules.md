# Module Availability

This table categorizes VTK modules based on their availability in the JavaScript API, indicating whether the classes are marshalled, or ignored for some reason.

The data is sourced from the VTK GitHub repository, where the status of each module is tracked in three categories: [auto-marshalled](https://github.com/Kitware/VTK/blob/master/Utilities/Marshalling/VTK_MARSHALAUTO.txt), [ignored](https://github.com/Kitware/VTK/blob/master/Utilities/Marshalling/ignore.txt), and [manually marshalled](https://github.com/Kitware/VTK/blob/master/Utilities/Marshalling/VTK_MARSHALMANUAL.txt).

> Use search and status filters to quickly find module availability.
